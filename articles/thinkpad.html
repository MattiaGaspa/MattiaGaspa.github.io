Thinkpad experience

28/01/2022

Some days ago I found an old Thinkpad T41 that my dad found a lot of time ago in a dumpster. The original project was to install gentoo, but I later discovered that it was a single core.

Holy sh*t: it would have taken me a lifetime, without considering that it could have crashed. But let's start from the beginning...

Hardware

The pc is an old Thinkpad T41. All components are original and the specs are:

  • Core: an Intel Centrino, single core at 1.6 GHz clock;

  • RAM: 1 Gb (ddr2);

  • GPU: an AMD ati rv250;

  • Disk: 71.2 Gb of HDD (connected with PATA).

OS

At first I was undecided between:

  1. Debian: but then I discarded it because it uses systemd (which is obviously bloated for this system);

  2. Void: because it uses runit, which is lighter than systemd. But I wanted to go further;

  3. Alpine: the chosen one, because it use OpenRC, my favourite init system, and it uses musl, a lightweight alternative to glibc even if it is not as complete as the latter.

During the automatic installation of Alpine Linux, using the setup-alpine command, I had a system crash because, after the root partition format, the system couldn't mount properly mount the partion on /mnt.

I had to do a semi-automatic installation so, after setting the various things, I manually partitioned and formatted the disk, mounted it and BOOM Alpine was installed! (after running setup-disk /mnt)

Features

The system uses an 100M ext4 boot partition, a 70G jfs root partition and a 1.2G swap partition. I have chosen a jfs filesystem for the root partiotion because I have read that use less system resources than zfs or ext4.

For desktop usage I have tried a lot of window manager, like ratpoison, swaywm, i3wm and icewm, and the best was icewm. Ratpoison often crashed while i3wm and swaywm was slower than icewm, the only cons of icewm is that I have never used it and the default configuration and themes sucks.

With flatpak installed I have been able to install telegram-desktop but not all applications are available (not and amd64 architecture).

Issues

The wi-fi doesn't work and installing linux-firmware didn't work. I haven't tested the other peripherals since I haven't used them in my life (what the f*ck does an infrared interface does on a pc???).

Final thought

19/02/2022

The laptop is pretty usable for office, LibreOffice works perfectly. Using firefox is not the best idea, but I think that other web browser can work pretty well. IMO this pc can be a nice firewall for the home network (of course not the best).

Update

After a reinstall, I changed the filesystems

  1. /dev/sda1 (boot partition): is now a vfat partition;

  2. /dev/sda3 (root partition): is now an xfs partition, a more performing filesystem.

I also replaced Firefox with netsurf browser, which is MOOOOOOOOOOOORE lightweight.

Unfortunately, while I was using the zathura pdf reader, the system crashed and I had to force shutdown throught poweroff button. After that the system gave an Initramfs error...

I tried to reinstall Alpine Linux but also the live ISO got the same error. Fortunately sometimes the system boot up properly so, when I will have time, I will reinstall the whole thing.
